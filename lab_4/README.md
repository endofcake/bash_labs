# Useful commands

A somewhat random assortment of commands that are essential for day-to-day work:

* `ls`
* `touch`
* `cd`, `pushd`, `popd`
* `cp`, `mv`
* `history`:
    * `!!` — run last command
    * `!*n*` — repeat command number *n*
* `tail`, `head`
* `grep`
* `less`
* `ps`
* `cat`
* `echo`
* `which`
* `find`
* `curl`, `wget`
* `vim`
